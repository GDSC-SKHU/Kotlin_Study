# Kotlin_Study
자바만 써본 사람들의 코틀린 공부

## Convention
자신의 이름으로 브랜치 만들고 해당 브랜치에서 과제 수행 후 PR  
모임할 때 Review 및 Merge <br>

자신의 이름으로 작성된 폴더 내에 주차별 폴더 생성  
예시) 프로젝트파일\방예혁\1주차\(과제 추가)
<br />
|Info|Commit Message|
|------|---|
|추가|Add: 메시지 내용(#주차별 이슈)|
|삭제|Delete: 메시지 내용(#주차별 이슈)|
|수정|Fix: 메시지 내용(#주차별 이슈)|  

메시지 내용은 한글로 작성  
예시) Add: 코틀린 기본 문법 정리(#1)
